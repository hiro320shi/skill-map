# スキルマップ

## レベル定義
- 0: 未経験
- 1: 初歩勉強中（チュートリアル・入門レベル）
- 2: 1人でできる（基本的な実装〜テストまで自走可能）
- 3: 経験豊富（後輩指導・レビューもできる）
- 4: 専門家レベル（チームをリードできる）
